## `$ sudo apt install linux-command-center`

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/linux-command-center.git
cd linux-command-center

# Install dependencies (tkinter usually pre-installed)
sudo apt install python3-tk  # Debian/Ubuntu
sudo pacman -S python-tk      # Arch

# Launch
python3 linux_command_center.py
```

> ⚡ No pip dependencies required. Pure Python stdlib.

<!-- USAGE -->
## `$ man lcc`

```
USAGE
  python3 linux_command_center.py

NAVIGATION
  • Click a category from the left sidebar
  • Click any command button to execute
  • Use the search bar to filter commands globally
  • Type custom commands in the bottom input bar
  • Press ENTER or click [▶ Run] to execute
  • Use ↑ / ↓ arrows to browse command history
  • Press TAB for path autocomplete

OUTPUT
  • Terminal output appears in the green-on-black viewer
  • [Copy Output] — copies entire terminal to clipboard
  • [Save Output] — saves to ~/lcc_output_TIMESTAMP.txt
  • [Clear] — clears the terminal
  • [■ Stop] — kills the running process
```
